RADA

A 3 DoF 3D first-person shooting game designed for Rokid Glass

My Role
  • Game Designer
  • Unity Developer

Team

Time

Platform

Lightweight and comfortable with a lasting battery, Rokid Glass is a standalone AI-powered augmented reality solution purpose-built for enterprises.
The mass production version was released in May 2019 and is available for order now.

Design Process

Design Goal

Demonstrate the hardware performance and capability.
Allow the user to quickly learn the usage of Rokid Glass through gaming.

Initial Idea and Sketch

We imagine that the AR glasses is a portal to another parallel universe that one wouldnâ€™t be able to see with naked eyes. AR glasses equip human with superpower to see more. With this concept in mind, we hope to create an adventurous experience in an unknown universe for the player. We designed a tunnel that connects the real world with the parallel universe, two types of aircrafts as attackers. The sketch shows some explorations for the morphology and transformation of the aircrafts and tunnel.

Game Components

Interactions

Considering the hardware capabilities and the goal of the intuitive design, below is the user interaction we defined.

swipe forward

to fire the machine gun

double tab

to fire the barrage when energy is fully charged

head turning

incorporating with the IMU sensor, the user can search and aim at targets
by turning his/her head.
