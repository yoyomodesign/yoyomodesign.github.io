Search Engine for Routers

A 2-day UX design challenge for a web application.

Exercise Requirements
  • Design a search engine for routers on a web application to help users easily locate one device they are looking for. A router has many properties, like router name, group name, status, IP address etc. When user types certain keyword, this search engine should be able to show users search results that include this keyword in a clear and comprehensive way.
My Role
  • UI/UX Designer
  • Used GUCDR design model to analyze and deduce the UX design strategy. Created detailed UI layouts and design specs.
