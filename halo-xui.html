Rokid Halo XUI Design Guidelines

A UI/UX design system for AR platform

Ideation

We started brainstorming from the three perspectives: environment, user input and UI elements.

The guideline was categorized into 4 sessions which are concept, spatial design, interaction, UI based on the above perspectives.

Exploration and Testing

We explored and created multiple versions of different design style, and had them tested on Rokid Glass.
After that, we designed questionnairs and conducted interal survey to collect feedbacks.

The following are examples of explorations.

Halo Design Principle

Based on the exploration, testing and findings, we ended up with these keywords as inspiration: universe, galaxy, futuristic. We also concluded the following main design principle.

Firstly, no large area of solid color is allowed. Some components need to be hollowed out for users to see more real world view.

Secondly, in order to avoid conflicting with real world light source, we should avoid adding dirctional light source to the design. Let the elements light themselves up.

Thirdly, we defined a set of retrofuturism color themes.
